# Applied Econometrics in R

## Overview
This repository contains applied econometrics analyses completed in **R Studio** using **R Markdown**.  
The project focuses on analyzing real economic datasets and translating statistical output into meaningful economic interpretation.

The emphasis is on **reproducible research**, clear documentation, and applied data analysis rather than purely theoretical modeling.

---

## Objectives
- Apply econometric techniques to real-world economic data
- Perform data cleaning and transformation using modern R workflows
- Generate descriptive statistics and conditional summaries
- Communicate results through reproducible R Markdown reports
- Interpret statistical results in an economic context

---

## Tools & Technologies
- **R / RStudio**
- **R Markdown**
- **tidyverse**
- **wooldridge** (applied econometrics datasets)
